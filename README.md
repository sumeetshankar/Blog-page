# 📝 Blog Page

A modern and responsive **Blog Web Application** built using front-end technologies.  
This project demonstrates a clean UI for displaying blog posts with a responsive layout and simple navigation.

---

## 🚀 Features

- 📖 Display blog posts in a structured layout  
- 🎨 Clean and responsive UI design  
- 📱 Mobile-friendly interface  
- ⚡ Fast and lightweight  
- 🧩 Easy to customize and extend  

---

## 🛠️ Tech Stack

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactive functionality  
- **React.js** – Component-based development

## 💻 Installation & Setup

1. Clone the repository

git clone https://github.com/sumeetshankar/Blog-page.git


2. Navigate to the project directory


cd Blog-page


3. Install dependencies


npm install


4. Run the project


npm start


The application will run on:


http://localhost:3000
